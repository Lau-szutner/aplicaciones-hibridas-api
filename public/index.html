Documentación de la API BookTags

Descripción

Esta es una API RESTful para la gestión de libros y autores.

Endpoints disponibles

1. Registrar Usuario

Método: POST

Ruta: /auth/register

Descripción: Permite registrar un nuevo usuario.


POST http://localhost:3000/auth/register

Headers:
Content-Type: application/json

Body (JSON):
{
    "username": "nuevoUsuario",
    "password": "contraseñaSegura"
}

2. Iniciar Sesión

Método: POST

Ruta: /auth/login

Descripción: Inicia sesión y devuelve un token JWT.


POST http://localhost:3000/auth/login

Headers:
Content-Type: application/json

Body (JSON):
{
    "username": "usuarioExistente",
    "password": "contraseña"
}

3. Obtener Todos los Libros

Método: GET

Ruta: /libros

Descripción: Obtiene una lista de todos los libros.


GET http://localhost:3000/libros

Headers:
Authorization: Bearer {token}

4. Filtrar Libros por Nombre

Método: GET

Ruta: /libros/filter

Descripción: Filtra los libros por nombre.


GET http://localhost:3000/libros/filter?search=nombre

Headers:
Authorization: Bearer {token}

5. Obtener Libro por ID

Método: GET

Ruta: /libros/:id

Descripción: Obtiene un libro por su ID.


GET http://localhost:3000/libros/{id}

Headers:
Authorization: Bearer {token}

6. Crear un Libro

Método: POST

Ruta: /libros

Descripción: Crea un nuevo libro.


POST http://localhost:3000/libros

Headers:
Authorization: Bearer {token}
Content-Type: application/json

Body (JSON):
{
    "title": "Nuevo Libro",
    "author": "Autor Desconocido"
}

7. Actualizar un Libro

Método: PUT

Ruta: /libros/:id

Descripción: Actualiza la información de un libro existente.


PUT http://localhost:3000/libros/{id}

Headers:
Authorization: Bearer {token}
Content-Type: application/json

Body (JSON):
{
    "title": "Título Actualizado",
    "author": "Autor Actualizado"
}

8. Eliminar un Libro

Método: DELETE

Ruta: /libros/:id

Descripción: Elimina un libro por su ID.


DELETE http://localhost:3000/libros/{id}

Headers:
Authorization: Bearer {token}

9. Obtener Libro por Título

Método: GET

Ruta: /libros/title/:title

Descripción: Obtiene un libro por su título.


GET http://localhost:3000/libros/title/{title}

Headers:
Authorization: Bearer {token}

10. Obtener Todos los Autores

Método: GET

Ruta: /authors

Descripción: Obtiene una lista de todos los autores.


GET http://localhost:3000/authors

Headers:
Authorization: Bearer {token}

11. Crear un Autor

Método: POST

Ruta: /authors

Descripción: Crea un nuevo autor.


POST http://localhost:3000/authors

Headers:
Authorization: Bearer {token}
Content-Type: application/json

Body (JSON):
{
    "name": "Nuevo Autor"
}
